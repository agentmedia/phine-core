= About The Core =

The core contains the basic elements of the Phine CMS.

== Version History ==

=== 1.0.0 ===

First usable version including the following elements.

* containers
* layouts
* websites & pages
* module & template logic
* backend users, groups & access rights

=== 1.0.1 ===

Added page url in database and a related form part. It can be used for modules
that need a page selection.

=== 1.0.2 ===

Added frontend members and groups. These serve to protect frontend pages 
or content elements.

=== 1.0.3 ===

Added content wording. It can be used to invidually override translations used in a 
content element. Still, bundles can (and should) feed the php translator with default translations.

== 1.0.4 ==

Added sitemap functionality and settings

== 1.1.0 ==

Added logging in database
